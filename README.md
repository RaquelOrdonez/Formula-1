# Formula-1
Formula 1 Race Predictor

## Usage

To set up the environment run the following command and follow the instructions:

```
$ source setup.sh

```

To install the library:

```
$ # pip install formula_1
$ pip install .
```

## Development

To install a development version of the library:

```
$ pip install -e .[dev]
```
